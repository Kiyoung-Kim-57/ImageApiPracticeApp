# ImageApiPracticeApp
이미지 리스트 성능 향상을 위한 테스트 프로젝트
<br>
*Unsplash api를 사용하고 api key는 "ApiList" 이름의 plist를 만들어 개인 api key를 넣어 사용
<br><br>
Test Cases<br>
1. 단순 AsyncImage 사용
2. 이미지 리사이징 사용
3. 이미지 리사이징 백그라운드에서 진행(async/await)
4. 이미지 다운샘플링
5. 이미지 다운샘플링 활용
